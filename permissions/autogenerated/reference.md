`vnidrop-fs:scope`

</td>
<td>

An empty permission you can use to modify the global scope.

## Example

```json
{
  "permissions": [
    "vnidrop-fs:all-without-delete",
    {
      "identifier": "vnidrop-fs:scope",
      "allow": [
        "$APPDATA/documents/**/*"
      ],
      "deny": [
        "$APPDATA/documents/secret.txt"
      ]
    }
  ]
}
```
